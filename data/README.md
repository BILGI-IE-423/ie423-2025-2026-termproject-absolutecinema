# Data Folder Instructions

Due to GitHub file size limits, the raw dataset may not be included in every clone of this repository.

1. Download **IMDB Dataset.csv** from [Kaggle — IMDB Dataset of 50K Movie Reviews](https://www.kaggle.com/datasets/lakshmi25npathi/imdb-dataset-of-50k-movie-reviews).
2. Place the file inside `data/raw/` using exactly this name: `IMDB Dataset.csv`
3. After running `scripts/02_preprocess_data.py`, cleaned data will appear in `data/processed/cleaned_imdb_reviews.csv`.

## File naming
- **Raw:** `data/raw/IMDB Dataset.csv`
- **Processed:** `data/processed/cleaned_imdb_reviews.csv` (generated by the preprocessing script)
